
# Dean Kawasaki - Pseudospectral RK4 PDE Solver

This project is a numerical integration of partial differential equations (PDEs) using a pseudospectral approach and the Runge-Kutta 4th order (RK4) method.
The code is part of the research for the paper
**_Impact of fluctuations on particle systems described by Dean-Kawasaki-type equations._**
Under review in *Physical Review E*.
